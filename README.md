# redo-rs

WARNING: You probably shouldn't use this. It's an experimental port that I got Claude to write to see how it did. I've not used it in anger, I'm not especially likely to any time soon. It's purely a curiousity project.

That being said, if you're interested in giving it a whirl, I'd be interested in hearing how you find it. It's a rust port of [apenwarr's implementation of Daniel J. Bernstein's redo build system](https://github.com/apenwarr/redo).

I've always liked the idea of redo but don't really love depending on code that is stuck in Python 2 and hasn't been updated in 7 years. I jokingly suggested getting Claude to rewrite it in Rust and then went "...actually, why don't I just do that?". So I just did that. This is the end result. The best I will say about it is that it seems to work just fine based on some very half-hearted testing.
